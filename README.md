# EventEase - Event Management Application

## Project Overview
EventEase is a Blazor WebAssembly application built with .NET 8 for managing events, user registrations, and providing a personalized dashboard experience.

## Features
- **User Authentication**: Simple name/email based login system
- **Interactive Dashboard**: Personalized dashboard with statistics and registrations
- **Event Management**: Browse, view details, and register for events
- **Real-time Updates**: Live state management across components
- **Form Validation**: Comprehensive validation using DataAnnotations
- **Data Export**: Export registrations to text files
- **Responsive Design**: Mobile-friendly interface

## Technology Stack
- **Frontend**: Blazor WebAssembly (.NET 8)
- **Styling**: Bootstrap 5 + Custom CSS
- **State Management**: Custom AppState service
- **Validation**: DataAnnotations with client-side validation
